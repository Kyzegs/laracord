# Webhooks

This section contains all methods related to Webhooks.

## createWebhook

Create a new webhook for a channel.

```php
public static array createWebhook()
```

### Returns

Returns `array`

### Example

```php
use Kyzegs\Laracord\Facades\Laracord;

 = Laracord::createWebhook();
```
